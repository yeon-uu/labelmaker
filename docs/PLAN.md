# PLAN — labelmaker

## 컨셉 (2026-07-04 확정)
화면 중앙에 파스텔 라벨프린터 기기가 있고, 온스크린 키보드 클릭 또는 실제 키보드 입력으로 문구를 치면 LCD에 표시된다. 장식 글리프(곰, 딸기, 클로버, 하트 등 모노크롬 라인아트)를 끼워 넣을 수 있고, 출력 버튼을 누르면 라벨이 슬롯에서 주르륵 올라오는 애니메이션 후 "완성" 표시 → 투명배경 사각 라벨 PNG 다운로드.

## 목적
1차: 인터랙티브 프론트 연습 + 배포 경험. 2차: 배포 후 소액 수익 실험 (기대치 낮음, 추정).

## 결정 기록
- **현실 기기 감성 채택, 픽셀 스타일 기각** — 라벨 출력물은 실사용(인쇄) 스티커 느낌이어야 하는데 픽셀 기기와 톤이 충돌. 한글 픽셀폰트 제약, GPT 이미지 생성이 픽셀보다 소프트 제품 렌더에 강한 점도 근거.
- **AI API 미사용** — 귀여움은 팔레트·패턴·타이포로 코드 생성. 비용 0원 유지.
- **기기 = GPT PNG + DOM 오버레이 하이브리드** — 본체는 이미지, 키보드 키와 LCD는 DOM으로 그려서 실제 눌림 애니메이션·텍스트 표시 구현.
- 기각 대안: 픽셀 펫 스터디 타이머(에셋 물량 과다, dyugame과 중복), 계절 다이어리 페이지(정적, 연습 가치 낮음).

## v1 스코프 (이것만)
- 기기 화면 중앙 배치, 온스크린 두벌식 키보드(클릭 입력, 한글 조합) + 실제 키보드 입력(hidden input + 네이티브 IME).
- LCD 미리보기 (텍스트 + 글리프).
- 글리프 팔레트 ~10종 (오리지널 모노크롬 SVG).
- 폰트 2~3종, 테이프 색 3종 (화이트/크림/핑크).
- 출력 애니메이션 → 완성 토스트 → 투명배경 PNG 다운로드 (Canvas 렌더).
